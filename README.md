הסטודיו של ישראל - אפליקציית כושר חכמה
למה בניתי את האפליקציה הזו?
הפרויקט הזה נולד מהשטח. בתור מאמן כושר,
הדבר הכי חשוב לי הוא לתת למתאמנים שלי שירות ברמה הכי גבוהה שיש ולעזור להם להתמיד ולהתקדם.

אחרי ששמעתי תלונות רבות מהמתאמנים שלי שGoogle SheetS זה דבר מסורבל מדי למעקב אחרי אימונים ומילוי תוצאות,
החלטתי שחייב להיות פתרון טוב יותר. רציתי להחליף את הטבלאות המעיקות בכלי נוח, מהיר וידידותי למשתמש.


פעם, פיתוח של אפליקציה כזו היה עולה לי עשרות אלפי שקלים.
היום, בזכות הכלים של Claude (Cursor) ו-Antigravity,
הצלחתי לבנות בעצמי דברים שבעבר רק חלמתי עליהם. 
האפליקציה הזו היא ההוכחה שאפשר להנגיש טכנולוגיה מתקדמת לטובת כושר ובריאות, בלי להסתבך.

מה יש באפליקציה?
חוויית משתמש פשוטה: סוף לעבודה עם טבלאות מסורבלות בזמן האימון.
מעקב התקדמות: מילוי תוצאות מהיר שמאפשר למתאמן לראות את השיפור שלו בזמן אמת.
כניסה מאובטחת: חיבור קל דרך גוגל או אפל (iOS) כדי לשמור על פרטיות המידע.
ניהול אימונים אישי וזוגי: מותאם בדיוק לשיטת העבודה בסטודיו שלי.

הטכנולוגיה (מאחורי הקלעים)
למרות שאני לא מתכנת במקצועי, השתמשתי בכלים הכי חזקים בשוק כדי שהכל יעבוד חלק:
Next.js - לבניית הממשק.
Claude & Antigravity - המנוע שעזר לי להפוך את הרעיון לקוד עובד.

אבטחה: מערכת אימות נתונים מתקדמת.
